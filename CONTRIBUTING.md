# Contributing to CubeTrix-UOL

Thank you for helping build an ethical, open, and regenerative ecosystem.

## Code of Conduct
- Respect others.  
- Be transparent.  
- Collaborate constructively.

## How to Contribute
1. Fork → Branch → Edit → PR.  
2. Run `compliance_checker.py`.  (coming soon)
3. Include ethical impact notes.  
4. Tag @ethics-review for sensitive updates.

## Checklist
- [ ] ETHICS.md updated if needed.  
- [ ] Compliance passed.  
- [ ] Credit preserved.
